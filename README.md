# factomind-technologies.github.io

Static snapshot of the Factomind company homepage (https://factomind.io),
exported from the original WordPress site for hosting on GitHub Pages.

- Generated: 2026-05-29
- Source: WordPress (Astra theme + Elementor), crawled and made fully static.
- All links are root-relative, so this works at the org-root Pages URL and at the factomind.io custom domain.
- `.nojekyll` disables Jekyll so the wp-content/ asset folders are served untouched.
